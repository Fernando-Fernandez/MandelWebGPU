WebGPU Fractal Lab

This project is a playground for experimenting with Mandelbrot renders using modern GPU APIs in the browser. Each demo explores different precision strategies and shader techniques to keep iteration counts high and zooms smooth while paving the way toward a WebGPU implementation.

Available Experiments

  • WebGL Mandelbrot (baseline) Reference implementation with rectangle zoom, smooth coloring toggle (s).
  • CPU emulated 64-bit Benchmarks emulated math paths to compare against native GPU approaches.

Getting Started

Serve the directory through a static web server (for example, python3 -m http.server) and open index.html in a browser with WebGPU enabled flags. From there, launch any experiment above and begin zooming into the set.
